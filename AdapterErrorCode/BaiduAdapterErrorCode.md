## 百度错误码

| 错误码  | 信息描述                                            |
| ------- | --------------------------------------------------- |
| 0       | 无广告返回                                          |
| 0103011 | 应用 ID 信息错误，MSSP 未收录                       |
| 0103060 | 应用包名信息错误，请保证注册包 名和实际请求包名一致 |
| 0107003 | 广告位 ID 与 APPID 不匹配                           |
| 1020001 | 网络连接失败                                        |
| 1040003 | 请求超时                                            |
| 1040001 | 请求时使用了错误的参数，比如使 用错误的广告位 ID    |
| 3030002 | 缓存物料失败                                        |
| 3040001 | 广告展现标准不达标                                  |
